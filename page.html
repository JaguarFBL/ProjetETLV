Code HTML

Cette page est une page de test, vous pouvez visulaiser son code-source HTML en faisant ctrl+U.
Vous pouvez aussi ouvrir l'inspecteur de code, qui vous permettra en plus de modifier le code pour faire des tests.
Ces modifications ne sont évidemment pas permanentes.

Vous verrez que le code-HTML comporte de nombreuses symboles spéciaux.
Ces symboles sont des éléments de code qui, heureusement, disparaissent une fois la page affichée.
